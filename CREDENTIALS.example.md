# Учётные данные для входа (пример)

**Не коммитьте файл `CREDENTIALS.md` в репозиторий.**  
Скопируйте этот файл в `CREDENTIALS.md` и заполните после первого развёртывания.

## После первой установки

1. Выполните сидер супер-администратора:
   ```bash
   php artisan db:seed --class=SuperAdminSeeder
   ```
2. Войдите по учётным данным из вывода команды (или см. [USER_ROLES.md](USER_ROLES.md)).
3. **Обязательно смените пароли** в профиле или через Tinker.
4. Заполните этот шаблон своими данными и сохраните как `CREDENTIALS.md` только локально (файл в .gitignore).

## Шаблон для CREDENTIALS.md

```
Супер-администратор:
  Email: _______________
  Пароль: (храните в надёжном месте)

Администратор:
  Email: _______________
  Пароль: _______________
```

Подробнее: [USER_ROLES.md](USER_ROLES.md).
